#Information_Security_Application

####Information_Security_project

###사용방법

```
1.누구나 알기 쉽게 어플리케이션으로 제작 하였다.

2.0~1023 까지 key 값을 입력하고

3. 암호화 시킬 평문 값을 입력한다.
4. 각각 암호화 복호화 버튼을 누르면 반응한다.

5. 초기화를 시키려면 초기화 버튼을 누른다.
```

###개발환경

```
Kotlin Language
Android Studio 사용 3.1.4 버전 사용
```
